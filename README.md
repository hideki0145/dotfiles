# dotfiles

## How to use

```sh
# Initialize of packages.
bash -c "$(curl -L https://raw.githubusercontent.com/hideki0145/dotfiles/master/etc/install.sh)" -s init
# Deployment of dotfiles.
bash -c "$(curl -L https://raw.githubusercontent.com/hideki0145/dotfiles/master/etc/install.sh)" -s deploy
```
